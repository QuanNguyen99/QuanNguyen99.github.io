Khóa Học và lộ trình

ĐỐI TƯỢNG HỌC VIÊN

  • Đam mê thiết kế, lập trình ứng dụng web, thương mại điện tử, landing page.
  • Có tư duy Visual Thinking - tư duy mỹ thuật trực quan.
  • Sinh viên, người đi làm trái nghành muốn trở thành lập trình viên web.
  • Sinh viên CNTT năm cuối làm đồ án tốt nghiệp web front end.

NỘI DUNG CHÍNH

  • Thiết kế giao diện UI và phân tích , tối ưu trải nghiệm người dùng.
  • Cắt giao diện từ file PSD, Sketch, Figma.
  • Lập trình JavaScript, jQuery kết hợp CSS, Bootstrap để thiết kế responsive web.
  • Lập trình JavaScript ES6.
  • Lập trình ứng dụng Single Page App với ReactJS.
  • Hoàn thành 3 đồ án môn học: thiết kế giao diện, lập trình web site hoàn chỉnh và ứng dụng Single Page App với ReactJS.
